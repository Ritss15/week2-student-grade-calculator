# Week 2 – Student Grade Calculator

## Project Overview
This project calculates student grades based on marks using Python.

## Features
- Uses if-elif-else conditions
- Input validation (0–100)
- Uses functions
- Displays encouraging messages

## How to Run
1. Install Python
2. Run:
   python grade_calculator.py

## Grading Logic
A: 90-100  
B: 80-89  
C: 70-79  
D: 60-69  
F: Below 60
